Pour créer le conteneur
    $docker run -t -i --name mail mail

Pour lancer le conteneur : 
    docker start mail

Pour aller dans le conteneur :
    docker exec -ti mail /bin/bash

Pour voir l'adresse IP du conteneur : 
    docker inspect mail | grep IP
